# NETFLIX-DATA-ANALYSIS-DASHBOARD-

Built a multi-layer Power Bi dashboard to analyze Netflix's content expansion patterns using Power Query and DAX.

KEY FINDINGS:-

1. TOTAL TITLES ANALYZED: 8,734
2. TV Show additions increased from 54 (2015) to 398 (2016), marking a structural acceleration phase.
3. Both Movies and TV Shows peaked in 2019:
4. Movies: ~3.1K additions
5. TV Shows: ~1K additions
6. Content growth expanded aggressively between 2016–2019, followed by a decline post-2019.
7. United States led production in 2019 with 955 titles, followed by India with 529 titles.
8. TV-MA emerged as the dominant rating category.

>Genre analysis shows:
1. Drama leading TV content
2. Romantic and Thriller dominating Movies

>Technical Implementation:
1. Data transformation using Power Query
2. DISTINCTCOUNT-based measures to prevent duplication bias
3. Time-series modeling using Year_Added
4. Interactive filtering
5. This project progressed from descriptive analytics to structural growth pattern analysis and strategic interpretation.
